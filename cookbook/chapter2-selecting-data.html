  • pick variables based on their names

  • pick rows based on their values

  • aggregate data (counting) by groups

2.1 Import data

Were going to use a new dataset here, to demonstrate how to deal with larger datasets. This is a subset of the of 311 service requests from NYC Open Data.

2.4 Whats the most common complaint type?

  • The base way
count_complaint <- tapply(complaints$`Unique Key`, complaints$`Complaint Type`, length)
## or: table(complaints$`Complaint Type`)
print(class(count_complaint))
#> [1] "array"
## noted that the output is an array, you can treat it like a vector (i.e sorting it)
count_complaint <- count_complaint[order(count_complaint, decreasing = TRUE)]
count_complaint[1:10]
#>                HEATING   GENERAL CONSTRUCTION Street Light Condition 
#>                  14200                   7471                   7117 
#> DOF Literature Request               PLUMBING        PAINT - PLASTER 
#>                   5797                   5373                   5149 
#>       Blocked Driveway               NONCONST       Street Condition 
#>                   4590                   3998                   3473 
#>        Illegal Parking 
#>                   3343
  • The data.table way
count_complaint2 <- complaints[, .(n = length(`Unique Key`)), by = .(`Complaint Type`)]
count_complaint2 <- count_complaint2[order(n, decreasing = TRUE)]
print(count_complaint2)
#>               Complaint Type     n
#>   1:                 HEATING 14200
#>   2:    GENERAL CONSTRUCTION  7471
#>   3:  Street Light Condition  7117
#>   4:  DOF Literature Request  5797
#>   5:                PLUMBING  5373
#>  ---                              
#> 161:       Open Flame Permit     1
#> 162:                     DWD     1
#> 163:  Highway Sign - Damaged     1
#> 164:            Ferry Permit     1
#> 165: X-Ray Machine/Equipment     1
## or more concise
count_complain3 <- complaints[, .(n = .N), by = .(`Complaint Type`)][order(n, decreasing = TRUE)]

ggplot(count_complaint2[1:10], aes(reorder(`Complaint Type`, n), n)) +

    coord_flip()

2.5 How it works

The basics of working with data.table are quite simple:

[I took the image from the slide of Arun Srinivasan]

which means: take data.table dt, subset rows using i and manipulate columns with j, grouped according to by. This syntax allows you to combine several operations in a very concise and consistent expression.

For example, the following line can be read as count the number of Unique Key per Complaint Type.

complaints[, .(n = length(`Unique Key`)), by = .(`Complaint Type`)]
